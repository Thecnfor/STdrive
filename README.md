# STM32 模块驱动库

这是一个收集各种 STM32 模块驱动代码的仓库。旨在方便开发者快速复用和集成常用的硬件模块驱动，减少重复造轮子的时间。

## 目录结构

目前仓库包含以下模块：

- `mqtt/`: MQTT 协议连接相关驱动
- (后续添加的模块将按文件夹分类存放)

## 贡献指南

欢迎提交新的模块驱动或优化现有代码！

### 代码贡献标准

为了保持代码风格一致并明确版权/责任人，请在每个源文件（`.c` / `.h`）的顶部添加以下标准注释头。

**注释模板：**

```c
/**
  * @file    [文件名]
  * @author  [开发者姓名/GitHub ID]
  * @version V1.0.0
  * @date    [YYYY-MM-DD]
  * @brief   [文件功能简要说明]
  *
  * -----------------------------------------------------------------------------
  * 修改历史:
  * [YYYY-MM-DD] [开发者姓名]: 创建文件
  * [YYYY-MM-DD] [开发者姓名]: [修改内容描述]
  * -----------------------------------------------------------------------------
  */
```

### 提交建议

1. **模块化**：请尽量将每个驱动模块放在独立的文件夹中。
2. **注释**：代码中请包含必要的中文注释，解释复杂的逻辑或硬件配置。
3. **接口**：建议提供清晰的 `Init`, `Read`, `Write` 等标准接口。

## 使用说明

请根据具体模块文件夹下的说明进行集成。通常只需要将对应的 `.c` 和 `.h` 文件添加到您的工程中，并根据需要修改硬件相关的宏定义或配置。
